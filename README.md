# hack-hate-util

Utilities and resources for our 2020 [Hack Hate](https://hackhate.org) event.

![Hack Hate logo](https://github.com/PoliceRewired/hack-hate-util/raw/main/HackHateTitle.png)

* `index.html` - the landing page for our domain, hackhate.org - redirects to the event pages.
* `project-graph/` - a tool for mapping the relationships between our participants' projects.
